# Skill: Refactor With ADR

## Quando usar

Refactors arquiteturais que alteram boundaries, contratos ou pipeline.

## Passos

1. Criar ADR em `docs/adr/`.
2. Definir migração incremental e compatibilidade.
3. Implementar por fases com checkpoints verdes.
4. Atualizar `docs/architecture.md` e READMEs impactados.
