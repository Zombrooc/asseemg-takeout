# Skill: Add Feature With Tests

## Fluxo obrigatório

1. Descrever feature e risco no PR.
2. Implementar em fatia pequena.
3. Adicionar teste unitário e/ou integração.
4. Atualizar docs se contrato/interface pública mudar.
5. Validar:
   - `pnpm run lint`
   - `pnpm run typecheck`
   - `pnpm run test`
   - `pnpm run build`
