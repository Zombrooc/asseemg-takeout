# Rule 030: Style System (Mobile)

## Política

1. Stack oficial de UI mobile: HeroUI Native + Uniwind/NativeWind.
2. Tokens/tema devem ser centralizados e reutilizados.
3. Evitar estilos duplicados e divergentes entre telas equivalentes.

## Validação

1. Mudanças de estilo base devem atualizar documentação em `docs/react-native.md`.
